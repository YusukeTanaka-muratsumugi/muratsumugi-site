# ムラツムギ公式サイト 運用マニュアル

このサイトは「**コードを触らずに更新できる**」ことを最優先に作られています。
日常の更新はすべて、①ブラウザの管理画面、または ②Claudeへの指示、のどちらかで完結します。

---

## 1. サイトの仕組み(1分で分かる版)

- サイトの**見た目(デザイン)**と**中身(文章・データ)**が分離されています。
- 中身はすべて `content/` フォルダの中の7つのファイルに入っています。

| ファイル | 内容 | 対応ページ |
|---|---|---|
| `content/news.json` | お知らせ記事 | トップ・お知らせ |
| `content/members.json` | メンバー・アドバイザー | メンバー |
| `content/activities.json` | 活動内容 | トップ・活動内容 |
| `content/works.json` | 主な実績 | 主な実績 |
| `content/about.json` | 私たちについて 本文 | 私たちについて |
| `content/message.json` | 代表メッセージ | 代表メッセージ |
| `content/site.json` | キャッチコピー・noteURL・フォームURL・団体概要 | 全ページ |

- この7ファイル(と写真)だけを更新すれば、サイトは自動で最新になります。
- `assets/` フォルダ(デザイン)や `.html` ファイルは、日常運用では**触りません**。

---

## 2. 日常の更新①:管理画面から(いちばん簡単)

サイト公開後、`https://サイトのURL/admin/` にアクセスすると管理画面が開きます。

### お知らせを追加する
1. `/admin/` にログイン
2. 左メニュー「お知らせ」→「お知らせ一覧」
3. 「記事を追加」を押し、ID・日付・カテゴリ・タイトル・本文を入力
   - IDは半角英数で、他と重複しないもの(例:`news-2026-08-workshop`)
   - 本文は、**空行を入れると段落が分かれます**。URLを書くと自動でリンクになります。
4. 「保存」→「公開」を押す
5. 1〜2分後にサイトに反映されます

### メンバーを追加・変更する
1. 左メニュー「メンバー」
2. 「主要メンバー」または「アドバイザー」の一覧で追加・編集
3. 顔写真はその場でアップロードできます(正方形に近い写真がきれいに表示されます)
4. 「保存」→「公開」

### そのほか
- 活動内容の文言修正 → 「活動内容」
- 主な実績の追加・編集 → 「主な実績」
- キャッチコピーやnote・フォームのURL変更、団体概要 → 「サイト基本情報」
- 「私たちについて」「代表メッセージ」の本文 → 「固定ページ」

## 3. 日常の更新②:Claudeに指示する

このサイトのGitHubリポジトリをClaude Code(またはClaude DesktopのCodeタブ)で開けば、
日本語で指示するだけでClaudeが更新できます。

指示の例:

- 「アドバイザーに○○大学の△△先生を追加して。専門は環境社会学」
- 「2026年度の実績に、□□町の集落調査業務を追加して」
- 「トップのキャッチコピーを『……』に変えて」
- 「お知らせに、9月のワークショップ開催告知を書いて追加して。日時は……」

Claudeは `content/` の該当ファイルを編集し、変更をコミット(保存)します。
GitHubに反映されると、サイトは自動で更新されます。

デザイン自体の変更(色を変えたい、ページを増やしたい等)も同じ方法でClaudeに頼めます。
その場合は `assets/css/style.css` や `.html` をClaudeが編集します。

## 4. Googleフォーム(お問い合わせ)の設定

1. Googleフォームで問い合わせフォームを作る
   (項目例:お名前/ご所属/メールアドレス/お問い合わせ内容)
2. フォーム右上の「送信」→「< >」タブを選ぶ
3. 表示されるHTMLの中の `src="https://docs.google.com/forms/d/e/…/viewform?embedded=true"` の
   **URL部分だけ**をコピー
4. 管理画面「サイト基本情報」→「お問い合わせフォームのURL」に貼り付けて公開

URLが未設定の間は、お問い合わせページに「準備中」と表示されます。

## 5. よくあるトラブル

| 症状 | 対処 |
|---|---|
| 保存したのにサイトが変わらない | 1〜2分待つ。それでも変わらなければブラウザを再読み込み(Ctrl+Shift+R) |
| 管理画面にログインできない | Netlify Identityの招待メールを確認。期限切れなら管理者が再招待 |
| 記事の段落が分かれない | 本文中の段落の間に**空行**(改行2回)を入れる |
| 写真が大きすぎて重い | 長辺1200px程度に縮小してからアップロードする |
| JSONを直接編集したら画面が真っ白になった | カンマの付け忘れ等の記法エラーの可能性。直前の変更を戻すか、Claudeに「content/○○.jsonの記法エラーを直して」と頼む |
